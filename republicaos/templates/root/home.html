O que é o Republicaos?

É uma ferramenta web para registrar e dividir as contas comuns entre os moradores de uma república de estudantes.

A ideia de desenvolver o Republicaos nasceu de uma necessidade pessoal de acabar de vez com o uso daquelas planilhas chatas e difíceis de manter que sempre sobrava para uma pessoa manter.
